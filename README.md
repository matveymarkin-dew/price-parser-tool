# Price Parser - Парсер цен конкурентов 💰

Автоматический мониторинг цен на товары с популярных маркетплейсов (Wildberries, Ozon, Яндекс.Маркет).

## Возможности

- 🔍 Парсинг цен с нескольких маркетплейсов
- 📊 Статистика: мин/макс/средняя цена
- 💾 Экспорт в JSON и CSV
- 📱 Уведомления в Telegram
- ⏰ Автоматический мониторинг по расписанию
- 📈 История изменения цен

## Установка

1. Установите Python 3.8+
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Для автоматического мониторинга установите дополнительно:
```bash
pip install schedule
```

## Использование

### Разовый парсинг

```bash
python parser.py
```

Программа запросит:
- Название товара для поиска
- Хотите ли отправить отчет в Telegram

### Автоматический мониторинг

1. Откройте `scheduler.py` и настройте:
```python
SEARCH_QUERIES = ["смартфон", "ноутбук"]  # Ваши товары
BOT_TOKEN = "ваш_токен"
CHAT_ID = "ваш_chat_id"
```

2. Запустите:
```bash
python scheduler.py
```

Мониторинг будет запускаться автоматически в 09:00 и 18:00 каждый день.

### Изменение расписания

В `scheduler.py`:
```python
schedule.every().hour.do(run_monitoring)  # Каждый час
schedule.every().monday.at("10:00").do(run_monitoring)  # Каждый понедельник
```
